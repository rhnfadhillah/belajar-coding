Catatan :
Tag pada head :
1. Judul : title
2. CSS : style
3. Java Script : script
4. Metadata : meta (untuk mendeskripsikan website)

Tag pada body :
1. Teks : h1, h2, h3, h4, h5, h6, p...
2. Pendukung teks : br (baris baru), hr (garis horizontal), em, strong...
3. Gambar : img
4. Hyperlink : a
5. list : ul, ol, li, dl, dt, dd
6. Tabel : table, thead, tbody
7. Form : form, input, select, button
8. Script : script
9. Object : object
10. Grouping : div, span
11. Komentar : <!-- -->

Struktur tag : <namatag atribut="nilai">
Contoh : 
<body bgcolor="lightblue">
<body bgcolor="lightblue" id="myid" class="myclass">

Atribut global:
accesskey, class, id, dir, lang, style, tabindex, title
